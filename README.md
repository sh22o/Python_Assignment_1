# Python Tasks Program

## Description
This Python script contains four small tasks that demonstrate basic programming concepts such as user input, arithmetic operations, conditional statements, loops, lists, and exception handling.

### Task 1: User Information
- Prompts the user to enter their name and age.
- Displays a greeting message with the entered information.
- Handles invalid age input using exception handling.

### Task 2: Calculator Operations
- Prompts the user to enter two numbers.
- Performs and displays:
  - Addition
  - Subtraction
  - Multiplication
  - Division

### Task 3: Grade Calculator
- Prompts the user to enter scores for three subjects.
- Calculates the average score.
- Assigns a grade based on the average:
  - **A:** 90 and above
  - **B:** 75–89
  - **C:** 60–74
  - **F:** Below 60

### Task 4: Number List Analysis
- Prompts the user to enter a list of numbers separated by commas.
- Calculates and displays:
  - Sum of numbers
  - Largest number
  - Smallest number
  - Average value

## Requirements
- Python 3.x

## How to Run

1. Save the script as `tasks.py`.
2. Open a terminal or command prompt.
3. Navigate to the folder containing the file.
4. Run the script using:

```bash
python tasks.py
```

or

```bash
python3 tasks.py
```

5. Follow the on-screen instructions and enter the requested values.

## Example Output

```text
Enter your name: Shereen
Enter your age: 25
Hello Shereen, you are 25 years old!
```

The program will then continue through the remaining tasks and display the corresponding results.
